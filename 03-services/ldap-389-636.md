## NMAP ENUMERATION

```bash
# Basic LDAP Enumeration
nmap -p 389 --script ldap-search,ldap-rootdse $ip

# Service & Script Scan
sudo nmap -n -sV --script "ldap* and not brute" -p 389 $ip

# Specific LDAP Scripts
sudo nmap -p 389 --script ldap-novell-getpass.nse,ldap-rootdse.nse,ldap-search.nse $ip

# LDAPS Scan
sudo nmap -p 636 -sV $ip
```
## CONNECT / BIND

```bash
# Anonymous Bind
ldapsearch -x -H ldap://$ip

# Simple Authentication
ldapsearch -x -H ldap://$ip -D "<user>@domain.local" -w '<password>'

# Authenticated with Base DN
ldapsearch -x -H ldap://$ip -D "<user>@domain.local" -w '<password>' -b "dc=domain,dc=local"
```
## BASIC ENUMERATION

```bash
# Get Naming Contexts
ldapsearch -x -H ldap://$ip -s base namingcontexts

# Base-Level Query
ldapsearch -h $ip -x -s base

# Dump All Objects (Anonymous if allowed)
ldapsearch -x -H ldap://$ip -b "dc=domain,dc=local"

# Retrieve All Attributes
ldapsearch -x -H ldap://$ip -b "dc=domain,dc=local" "*"
```
## USER ENUMERATION

```bash
# Enumerate Users
ldapsearch -x -H ldap://$ip -b "dc=domain,dc=local" "(objectClass=user)"

# Extract sAMAccountNames
ldapsearch -x -H ldap://$ip -b "dc=domain,dc=local" "(objectClass=user)" sAMAccountName
```
## GROUP ENUMERATION

```bash
# Enumerate Groups
ldapsearch -x -H ldap://$ip -b "dc=domain,dc=local" "(objectClass=group)"

# Get Group Members
ldapsearch -x -H ldap://$ip -b "dc=domain,dc=local" "(cn=Domain Admins)"
```
## LDAPDOMAINDUMP

```bash
# Basic Dump
ldapdomaindump $ip

# Authenticated Dump (LDAPS)
ldapdomaindump ldaps://$ip -u 'DOMAIN\user' -p 'password'
```
## SENSITIVE ATTRIBUTE SEARCH

```bash
# Search for Password-related Fields
ldapsearch -x -H ldap://$ip -b "dc=domain,dc=local" "(description=*)"

# Check pwdLastSet
ldapsearch -x -H ldap://$ip -b "dc=domain,dc=local" pwdLastSet
```

## NOTES

```md
- LDAP runs on TCP 389 (default)
- LDAPS runs on TCP 636
- Anonymous bind may be enabled
- Often integrated with Active Directory
- Naming Contexts reveal domain structure
- LDAP can expose usernames, groups, and descriptions
- Always check for misconfigured permissions
```
