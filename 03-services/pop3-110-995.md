## CONNECTION METHODS

```bash
# Plain POP3 (no encryption)
nc -nv <IP> 110
telnet <IP> 110

# POP3 with STARTTLS (upgrade to encrypted)
openssl s_client -connect <IP>:110 -starttls pop3 -crlf

# POP3S (implicit SSL/TLS)
openssl s_client -connect <IP>:995 -crlf -quiet
```

## POP3 COMMANDS REFERENCE

| Command | Description | Example |
|---------|-------------|---------|
| `USER <name>` | Identify user | USER john |
| `PASS <pass>` | Authenticate user | PASS P@ssw0rd |
| `STAT` | Get mailbox status (messages/total size) | STAT |
| `LIST [n]` | List messages or specific message | LIST / LIST 2 |
| `RETR <n>` | Retrieve message n | RETR 1 |
| `DELE <n>` | Mark message n for deletion | DELE 3 |
| `RSET` | Reset all deletion marks | RSET |
| `TOP <n> <lines>` | Show first X lines of message n | TOP 1 10 |
| `UIDL [n]` | Get unique-ID listing | UIDL |
| `CAPA` | List server capabilities | CAPA |
| `NOOP` | Keep connection alive | NOOP |
| `QUIT` | End session & apply changes | QUIT |

## COMMON RESPONSES & ERROR CODES

```
+OK = Command successful
-ERR = Command failed

Common errors:
-ERR Authentication failed  # Wrong credentials
-ERR No such message         # Message number doesn't exist
-ERR Mailbox locked          # Another session active
-ERR Protocol error          # Invalid command sequence
```
